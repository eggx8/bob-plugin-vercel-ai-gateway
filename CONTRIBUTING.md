# Contributing

感谢参与改进。提交 Pull Request 前请：

1. 保持插件运行代码无第三方依赖。
2. 使用 pnpm 安装开发环境并运行 `pnpm run check`。
3. 运行 `pnpm run package`，并提交更新后的 `.bobplugin`。
4. 若改变版本，保持 `package.json`、`info.json`、`appcast.json` 和 `CHANGELOG.md` 一致。
5. 不要在 issue、日志、测试或提交中包含真实 API Key。

Bug 报告应包含 Bob 版本、模型 ID、深度思考状态和已脱敏的错误信息。
